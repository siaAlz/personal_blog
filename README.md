# Personal Blog App with FastAPI & RestAPI 

## Features:
* Guest Section — A list of pages that can be accessed by anyone:

    1. Home Endpoint: This endpoint will return the list of articles published on the blog.
    2. Article Endpoint: This endpoint will return the content of the article along with the date of publication.

* Admin Section — are the pages that only you can access to publish, edit, or delete articles.

    1. Dashboard: This page will display the list of articles published on the blog along with the option to add a new article, edit an existing article, or delete an article.

    2. Add Article Page: This page will contain a form to add a new article. The form will have fields like title, content, and date of publication.

    3. Edit Article Page: This page will contain a form to edit an existing article. The form will have fields like title, content, and date of publication.
